产品力 - 产品经理能力评估模型

March 11, 2018

NOTE: 根据前人的分享与自己的总结，系统性的归纳一套产品经理能力评估模型。以便准确定位并提高自己。

硬实力

广度

战略布局（谋）
  • 打大仗
  • 做三级火箭
宏观视野（划）
  • 体：全球宏观经济体走势分析
  • 面：经济体内市场发展趋势分析
  • 线：市场内赛道分析
    • 规模计算
    • 当前格局
    • 发展趋势推演
  • 点：赛道切入点选择，切入策略制定
    • 中远期目标确定
    • 产品定位和发展方向确定
    • 分解行动路径，细化小目标
中观套路（术）

初阶

  • 输出能力：流程图、原型图、PRD
  • 需求理解与挖掘
  • 竞品分析与改进
  • 数据分析与改进
  • 反馈分析与改进

中阶

  • 用户画像
  • 整体流程图
  • 用户体验地图
  • 服务蓝图
  • AARRR模型
  • 病毒传播K因子
  • ABtest
  • 版本规划/节奏
  • 用户调研与分析
  • 成本控制
  • 风险控制

高阶

  • 冷启动方案设计
  • MVP产品方案设计
  • 种子用户收集与访谈
  • 产品从0到1的过程
  • 上线前后产品运营策略
微观感知（感）
  • 确定性、依赖
  • 愉悦、不爽、愤怒、恐惧、
  • 兴奋、不确定性、压力、防御心、嫌麻烦、焦虑
  • 成就感、归属感、刺激感、互动感、优越感、满足感、安全感、孤独感

深度

  • 增长、策略、商业化、国际化
  • 做核心指标、反作弊、活动策划、AI
  • 激励体系、会员积分体系、上瘾机制

软性素质

基础素质

  • 学习能力
  • 执行力
  • 沟通能力
  • 创造力
  • 用户体验
  • 设计审美
  • 同理心
  • 独立而深入的思考能力

关键素质

  • 团队融合
  • 行业融入感+主人翁精神
  • 心态和情商

关联知识

  • 技术知识
  • 项目管理
  • 财务、心理学、美学、办公技能等

市场能力

  • 对外商务沟通

运营能力

  • 品牌/公关/推广
  • 渠道管理

领导力

  • 团队能效
  • 方法论建设
  • 知识传承
  • 人才培养

个人势能

资源

  • 人脉关系
  • 产业资源
  • 融资能力
  • 政府资源
  • 资金实力
  • 团队构建能力

影响力

  • 个人品牌
  • 号召力

历史战绩

  • 打过的仗
  • 个人荣誉

腾讯PM段位划分

另外，附上腾讯PM段位划分。特别感谢梁宁的分享。受益良多。
（以下内容摘自梁宁 - 产品经理的段位）

一级PM

初级、核心工作就是优化特性，提升数据。
产出：优化现有产品特性。

  • 1.1 建立对产品的完整认知，懂得让一个产品持续提供服务的显性特性与隐形特性。而不是总被表面的显性特性所迷惑纠缠。
  • 1.2 告知需要优化的点，能够完成优化，实现产品数据表现的提升。
  • 1.3 告知需要提升的产品数据，主动判断通过优化哪些特性，实现数据提升。

二级PM

中级、定义产品并运营产品。
产出：一个能在市场中持续运营的产品。

  • 2.1 定义产品，控制产品范畴与整体服务成本。定义并打磨核心特性点。
  • 2.2 根据渠道特性，优化产品。取得规模用户。
  • 2.3 优化用户动线，巩固用户黏度。平衡收入与用户黏度。

三级PM

高级、定义并驾驭战略产品。
产出：创业项目，或者公司战略项目。

四级PM

大神、奇点探索者。
产出：神级产品，开启一个时代。
